Le Projet : Création d'une intelligence artificielle pour un jeu qu'on a créer sur java avec NetBeans

Nous avons un plateau où deux joueurs joue. Chaque joueurs ont des soldats avec des pv qui peuvent ce déplacer de gauche et droite ou haut et bas et attaquer un autre soldat si il est à côter (d'une case près).

  • Le déplacement se fait sous forme : XXDYY pour XX numéro de la case de départ et YY pour l'arrivé.
  • L'attaque se fait sous forme : XXDYYAZZ pour ZZ numéro de la case du soldat.

Plus tu attaque/élimine un soldat plus tu as des points.

Notre objectif était de créer une classe permettant de retourner toutes les possibilitées que le joueur peut faire sur un plateau. Cela devait évidamment respecter les règles dites précédamment.
Ensuite, nous avons du créer une intelligence artificielle qui sera amené à participer dans un tournoi. Ainsi nous avons du utiliser :

  • Java sur NetBeans
    • Java "classique" (Notion de classe, tableau, variable etc...)
    • Création de test unitaire sur NetBeans
  • Git avec gitlab
