Le Projet : Création d'un jeu en java

L'objectif de ce projet est de créer un jeu où on peut se déplacer dans un labyrinthe avec des monstres qui si tu touche, tu meurt. Un code initiale fut fourni avec l'interface graphique dedans. Dans ce jeu, il faut aussi créer le labyrinthe à partir de fichier texte à lire.

Les compétences apprises et utilisées :

  • Création du projet java sur NetBeans
    • JavaFX
  • Utilisation de gitlab
  • Création de diagramme de classe
