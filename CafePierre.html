Etude du réchauffement climatique utilisant une base de données sur python

L'objectif est de faire plusieurs graphiques pour savoir comment la pollution est repartie dans le monde. C'est-à-dire quelle sont les pays qui pollue le plus, qui subira les conséquences de cette pollution et à quelle point cette pollution change le climat.

Les compétences apprises et utilisées :

  • Python sur Visual Studio Code
    • avec Jupyter
    • Pour créer des diagramme :
      • geopandas
      • matplotlibpyplot
    • csv
    • sqlite3
    • pandas
    • datetime
  • Utilisation et création du base de donnée interne
    • Création depuis un .csv avec csv
    • avec sqlite3 (Utilisation et Création) et pandas (Utilisation)
  • Travail d'équipe (equipe de 5 personne)
  • Utilisation de Git sur gitlab
  • Gestion :
